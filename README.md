# Простий калькулятор
Навчальний проєкт для вивчення Git та Python.
## Опис
Простий консольний калькулятор, який виконує базові арифметичні
операції:
- Додавання (+)
- Віднімання (-)
- Множення (*)
- Ділення (/)
- Степінь (**)
- Остача від ділення (%)
## Запуск
Для запуску програми виконайте команду
python calculator.py або python3 calculator.py
